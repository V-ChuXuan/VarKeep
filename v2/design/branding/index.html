VarKeep

Windows environment variable snapshots. 六个图形方向，全部为单色矢量，优先保证 Windows 小图标尺寸的辨认度。

命名说明 Var 表示环境变量，Keep 表示保存和留存。名字不使用 Vault、Safe 或 Lock，避免误导用户认为项目提供加密。

Var 首字母 · 留存状态

非对称 V 表示变量，独立状态点表示已保存的快照。结构最简单，在任务栏和窗口标题的小尺寸下最稳定。
